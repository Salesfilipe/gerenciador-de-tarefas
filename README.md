# Gerenciador de Lista de Tarefas

Gerenciador de tarefas desenvolvido em Python durante a trilha da Rocketseat.

## Funcionalidades

- Adicionar tarefas
- Visualizar tarefas com status
- Alterar nome de uma tarefa
- Completar uma tarefa
- Excluir tarefas completadas

## Como usar

1. Execute o arquivo `gerenciador.py`
2. Escolha uma opção do menu digitando o número correspondente
3. Siga as instruções na tela

## Tecnologias

- Python 3

## Aprendizados

- Funções com `def`
- Listas e dicionários
- Laços com `for` e `while`
- Tratamento de erros com `try/except`
- List comprehension
